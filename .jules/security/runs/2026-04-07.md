This function is serializing an internal rust struct to serde_json::Value, which could theoretically fail. While this specific instance is in a test, removing panics in tests is part of the overall burn-down goal for the repository.

## Options

- Propagate the error using ? by changing the test return type to Result<(), Box<dyn std::error::Error>>.
- This fits the repo as it burns down an unwrap and prevents an unhandled panic.

### Option B

- This is not recommended because it still leaves a panic.

## Decision
