### Option A (recommended)
- What it is: Set `test = false` in `crates/tokmd-python/Cargo.toml` for the cdylib target.

- Trade-offs: Structure: Requires maintaining Python tests separate from cargo tests.

- Trade-offs: It complicates the Cargo features and can be fragile to configure correctly.
